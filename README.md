##  HBDetail.js
与LXK 的脚本可共存<br>
用来查看每个账户的今明后天各个平台红包详情<br><br>
<b>环境变量</b><br>
红包详情通知等级 HBDETAIL_NOTIFY_LEVEL<br>
0： 包含今明后过期详情， 完整通知<br>
1： 不包含详情， 只有总览（ 默认）<br>
